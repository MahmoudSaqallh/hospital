# Clinic App

واجهة عربية لإدارة رحلة المريض من استعراض العيادات إلى الحجز ومتابعة المواعيد.

## المزايا الحالية

- الصفحة الرئيسية مع Hero وبحث/تصفية التخصصات.
- دليل عيادات كامل (`/clinics`) مع ترتيب وتصفية.
- صفحة جدول تخصص ديناميكية (`/schedule/[id]`) مع حالات واضحة للمواعيد.
- صفحات معلومات أساسية: `/about`, `/contact`, `/privacy`.
- حجز مبدئي متعدد الخطوات (`/booking`) مع حفظ محلي للمواعيد.
- بوابة مريض (`/patient`) للبحث عن المواعيد برقم الجوال.

## البنية التقنية

- Next.js App Router + React + TypeScript.
- Tailwind CSS (v4).
- طبقة بيانات موحدة في `lib/data.ts` مع تحقق Zod عبر `lib/schemas.ts`.
- تتبع مشاهدات الصفحات عبر `/api/analytics`.

## التشغيل المحلي

```bash
npm install
npm run dev
```

التطبيق يعمل على: [http://localhost:3000](http://localhost:3000)

## الاختبارات

```bash
npm run test
```

## إرشادات التطوير

- تجنب استخدام `any` في مسارات البيانات الأساسية.
- استخدم `lib/data.ts` كمصدر موحد لبيانات العيادات والجداول.
- عند إضافة Route ديناميكي جديد، وفر حالات `not-found` و`loading` و`error` المناسبة.
- أي نموذج جديد يجب أن يضيف تحقق إدخال واضح ورسائل خطأ مفهومة بالعربية.
